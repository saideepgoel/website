COLOR PROPERTIES

  1. Text-Align:

      2. It is used to align the text. It can be used as: 

      text-align:center;

      And it will work as:

      TEXT

  2. Text-Decoration:

      3. It is used to give add lines in text. Its values are:

      1. underline
      2. overline
      3. line-through
      It can be used as: 

      text-decoration:underline overline line-through; (we can use multiple lines)

      And it will work as:

      TEXT
